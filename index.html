  2. Ruby on Rails Development Environment overview (lecture 3)
    • You will need to set up a development enviroment so you can build web apps using Ruby on Rails
    • You will need just one development environment for this
    • You will have multiple options for develoopment environments
      • Online cloud based IDE's - Browser Based IDE (Recommended for starter)
        • All your code is done in the cloud, using a browser based development environment
        • System independent, you could be using Mac, Windows, Linux doesn't matter
        • Installation of what we need is already present, very little to no setup required, simply sign-up and go
      • Local development environment (Not recommend for starter)
        • Your computer's file-system (Mac, Unix, Windows)
        • Requires installation of the framework and components that are neccecsary to develop code
        • Can have installation issues, cause frustration, will not work properly with Windows (cons)
  3. Introduction to Section 2 and Ruby language
    • What is ruby? - A dynamic, object-oriented programing language, designed to make programmer happy
    • Ruby Resource: www.ruby-lang.org
    • First ruby program - Hello World
      • Helloworld.rb - Thực hiện lệnh puts

        puts "Hello World"

      • Helloworldstring.rb - Thực hiện lệnh puts thông qua khai báo string

        hello = "Hello World!"
        puts "Hello World!"

      • Helloworldmethod.rb - Thực hiện bằng cách define method để puts string

        def greeting(hello)
        puts hello
        end
        greeting("Hello World!")
